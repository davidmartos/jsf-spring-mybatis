==================

Template for building JEE app with JSF, Spring and Mybatis
- Simple form
- Persistence
- Internationalization
- JSF handling exception

Frameworks
- JDK 8
- Tomcat 8 / Wildfly 8 / Jetty 9.2.3
- Spring IO Platform 1.1.2 (JSF 2.2, Spring 4.1, Hibernate 4.3)
- MyBatis 3.2
